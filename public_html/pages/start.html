First time setup

Install the Exactlink software with the provided serial code on your device.

Connect the Exactlink device with the provided cables and RS232 adapter. Connect the adapter to your machinetool and the USB cable to your tablet or PC.

When starting Exactlink, it connects to the device with default settings. Next, update your settings to your machinetool specifications. Press upload to confirm. You can find more item information by pressing the ? button in the bottom left corner.

Packaged items

  • Exactlink device
  • DB25 adapter
  • RJ45 cable
  • USB cable

Basic use

> Uploading files

Upload a file in just three clicks!
  • Open a file or create one from the default template.
  • Modify GCODE if necessary.
  • Upload!

> Downloading files

Download a file in just two clicks!
  • Start receiving by selecting:
    • Edit file after download.
    • Save file on your system.
  • Start sending from your machine.
  • If downloading does not automatically stop, cancel download. Everything received will be saved.

> Dripfeeding

Dripfeed files if they exceed memory capcacity.
If your machinetool has the Drip feed option, you'll be able to do so easily by:
  • Opening a file or create one from the default template.
  • Modify GCODE if necessary.
  • Select your transmit count, resend GCODE automatically for each part you manufacture.
  • Send! During transfer you can Cycle Start, Feed Hold, End repetitive uploading and toggle Single Block mode.

    Single Block mode or SBK will transmit GCODE and Feed Hold when it encounters a ; or new line. This can also be turned on by default in the settings menu.

It doesn't work?

Make sure Exactlink is up to date

Check if your Exactlink is running the latest software by connecting to the internet or visiting our software page.

Machine throws error when starting transmission

  • Make sure your transmission settings match your machine's Baudrate, Bits, Stop Bits and Parity.
  • Replug your RS232 adapter.
  • Disable Software Flow Control [SWF]

    If this solves your issue: Make sure to first set input mode on your machine. Proceed by sending from Exactlink.

Device throws error

  • Restart
  • Write down error code or take a picture of the screen
